# blog-feng
🔥node-express-MongoDB-typescript（后端与pcweb），taro（小程序），react全家桶（管理后台）；主要功能模块：博客，教程手册，电影等。
其中有简单的电影爬虫，用于定时抓取电影天堂数据；
